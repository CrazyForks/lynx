Lynx Gfx
========

Lynx Gfx provides shared graphics and animation primitives for Lynx on the Harmony platform.

## Overview

This module packages the native Lynx Gfx library as a Harmony HAR so other Lynx components can depend on the same graphics implementation.

## Installation

```bash
ohpm install @lynx/gfx
```

## How to use

You can add dependency in oh-package.json5 like this:

```json5
{
  "dependencies": {
    "@lynx/gfx": "0.0.1-alpha.1",
  }
}
```
